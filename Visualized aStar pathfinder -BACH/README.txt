This project visualizes the A* shortest path algorithm.

<need to install the SFML c++ libary to run the code>
<https://www.youtube.com/watch?v=lFzpkvrscs4&ab_channel=EOD-Ethan> // video on how to doso


=======================================================================================================
Controls:

Mouse-leftClick: set/remove block
Mouse-rightClick: set the start/target

KeyBoard-Enter: run the algorithm when done setting up the board
KeyBoard-BackSpace: clear the trace of past search after the algorithm

This program will be ready to receive  more input and rerun the algorithm after [BackSpace] is entered.
